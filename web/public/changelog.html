เวอร์ชันล่าสุด v0.1.14
v0.1.14
8 มิ.ย. 2026
ล่าสุด
  • เพิ่มรองรับเมาส์ทั่วทั้งแอป — คลิกแท็บเพื่อสลับหน้า · คลิกรายการเพื่อเลือกแล้วดับเบิลคลิก (หรือคลิกซ้ำที่รายการเดิม) เพื่อเปิด · หมุนล้อเพื่อเลื่อนดู
  • เพิ่มคลิกปุ่มในกล่องยืนยัน เลือกรายการในหน้าต่างซ้อน (command palette, ธีม, กระโดดไปบท, ส่งออก) และคลิกนอกหน้าต่างเพื่อปิดได้
  • เพิ่มในหน้าอ่าน (Reader) หมุนล้อเพื่อเลื่อนเฉพาะช่องที่เมาส์ชี้อยู่เมื่อปลดการเลื่อนพร้อมกัน (sync) ออก
  • แก้ไขบางครั้งคำแปล โดยเฉพาะช่วงบทแรก ๆ จะดึงประโยคจากบริบทต่อเนื่อง (continuity) มาซ้ำตอนต้น — ตอนนี้ระบบตัดส่วนซ้ำออกและแปลเฉพาะท่อนปัจจุบัน

  • แก้ไขจำกัดเหตุการณ์ของไปป์ไลน์ให้อยู่เฉพาะเล่มที่กำลังรัน ไม่รั่วไปยังเล่มอื่น

รีลีสแรก
  • เพิ่มรีลีสแรกของ honya — แอปเทอร์มินัล (TUI) สำหรับจัดโปรเจกต์แปลไลต์โนเวลญี่ปุ่น→ไทยผ่าน API ที่รองรับ OpenRouter
  • เพิ่มนำเข้า EPUB โดยเรียงตามลำดับสันหนังสือ ย้ายภาพประกอบ และล้าง HTML ให้เป็น Markdown
  • เพิ่มไปป์ไลน์สามเอเจนต์ — Orchestrator · Translator · Reviewer
  • เพิ่มTUI ห้าหน้าจอ — 書架 / 棚 / 訳 / 読 / 辞
  • เพิ่มเชื่อมต่อ OpenRouter พร้อมถามคีย์ของคุณตอนเริ่มต้นใช้งาน
